# HR-Analytics-Dashboard
## 1.	Project Title:
📊 HR Analytics Dashboard – Employee Attrition Analysis

An interactive Power BI dashboard designed to analyze employee attrition patterns, workforce demographics, and compensation trends. This project helps HR teams and business leaders identify key factors behind employee turnover and make data-driven decisions to improve employee retention.

## 2.	Description:
Developed an interactive HR Analytics Dashboard in Power BI to analyze employee attrition, workforce demographics, salary distribution, and job role trends. The dashboard leverages Power Query for data transformation and DAX for KPI calculations to provide insights into attrition patterns across age, education, salary slabs, and tenure, enabling data-driven HR decision-making and improved employee retention strategies.

## 3.	Tech Stack:
The dashboard was built using the following tools and technologies:

• 📊 Power BI Desktop – Data visualization and dashboard creation

• 📂 Power Query – Data cleaning and transformation

• 🧠 DAX (Data Analysis Expressions) – Creating calculated measures and KPIs

• 🔗 Data Modeling – Structuring relationships between HR datasets

• 🗄️ Microsoft Excel – Data storage, initial exploration, and dataset preparation

## 4.	Data Source:
### Source: Sample HR Employee Attrition dataset

• Employee demographics (Age, Gender, Education)

• Job role and department

• Salary and compensation details

• Years at company

• Attrition status

Total dataset includes 1,400+ employee records representing workforce information.

## 5.	Features / Highlights: 
### 🔎 Business Problem

• High employee attrition leads to increased hiring and training costs.

• Organizations struggle to identify the key factors causing employees to leave.

• Workforce data is often scattered, making it difficult for HR teams to analyze trends.

• Lack of clear insights prevents HR from taking proactive retention measures.

### 🎯 Goal of the Dashboard

To design an interactive hr analytics dashboard that:

• Analyze employee data to identify patterns and drivers of attrition.

• Provide clear insights into workforce demographics, salary distribution, and job roles.

• Help HR teams identify high-risk employee groups with higher turnover rates.

• Enable data-driven decisions to improve employee retention and workforce planning.
